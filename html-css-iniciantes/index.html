HTML

O que é o HTML?

HTML é o Hypertext Markup Language (HTML)

É a linguagem de marcação de hipertexto.

responsável por marcar o conteúdo e definir a estrutura do site.

Com ele podemos marcar os textos (Títulos, parafrafos, listas), adicionar imagens, vídeos, links e mais além de dar sentido (semântica) ao conteúdo.

Atrabés das marcaçãos das tags feitas com o HTML, conseguimos selecionar elementos para mudarmos o seu estilo ou comportamento: interação com CSS e Javascript.

As tags do HTML

As tags servem para marcamos o conteúdo no HTML. Geralmente abrimos e fechamos após o conteúdo.

As tags servem para inserirmos conteúdos como textos, imagens, vídeos e outros.

As tags também servem para dar informações para o browser como título do site, linguagem e outras.

O html é case insensitive, mas é boa prática escrever com minúsculas.

Um atributo dá informações extras sobre uma tag para o browser.

Importante lembras: Case insensitive o nome do atributo e case sensitive o valor do atributo.

Títulos - Heading

As tags h1, h2, h3, h4, h5 e h6 marcam diferentes níveis de títulos.

Paragráfos

a tag p macra um parágráfo.

Quebra de Linha

a tag br cria uma quebra de linha.

Link

A tag a marca um link.

Os links podem ter um caminho (path):

  • Absoluto
  • Relativo

Caminho Absoluto

Usado para arquivos externos ao nosso site. EX: uma URL.

Caminho Relativo

Usado para arquivos internos do site.

Semântica

A semnântica serve para dar sentido ao conteúdo, a escrita de um documento semântico beneficia principalmente leitores de tela(acessibilidade) e leitores de códigos(robôs como o do Google)


CSS

O que é o CSS?

É a linguagem de estilo.

Define o estilo do documento. Isto é, define a cor do texto, tamanho do texto, tamanho da imagem, poisção do conteúdo, cor de fundo, sombras e mais.

Teoria sobre CSS

Link: cria uma relação entre um documento HTML e um arquivo de estilo CSS.

rel: Define o tipo de arquivo (stylesheet para CSS). é possivel linkar outros também como favicons.

href: define o caminho do arquivo.

Seletores

Os seletores podem ser as tags:

  • h1
  • p
  • h1, p : a vírgula permite selecionar multiplis elementos para a aplicação de um mesmo estilo
  • p a > seleciona o a que tiver um o como elemento pai (não precisa ser filho direto)
Classes e IDS

Classe é chamada com .

Id é chamado com #

background e background-color

Muda a cor de fundo do elemento.

Um elemento h1, por exemplo, você pode colocar o background na cor black e o color na cor white.

Ja um elemento a, por exemplo, você pode colocar o background-color de seagreen e o color de white.

Div

A div é um elemento de bloco BLOCK genérico que serve para auxiliar no posicionamento dos elementos/conteudos na tela.

Existem também elementos semânticos como main, section, nav e outros que veremos adiante.


Javascript

O que é o Javascript?

Linguagem de script (programação.)

Altera o comportamento do documento. É utilizada para modificar o estilo, criar galerias de imagens, observar as ações do usuário, puxar informações de outros site, manipular dados e mais.
